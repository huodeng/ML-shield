数据集上传

本章将详细介绍如何在ML Security Simulator平台上传和管理您的数据集，以便进行后续的安全性分析。

支持的数据集格式

ML Security Simulator平台支持多种格式的数据集，包括：

  • CSV格式：结构化数据，适用于表格型数据集
  • 图像数据集：支持JPEG、PNG等常见图像格式，适用于计算机视觉模型
  • 压缩文件：支持ZIP格式的压缩文件，系统会自动解压并处理
  • 标准数据集：内置支持MNIST、CIFAR-10等常用基准数据集

上传数据集

通过Web界面上传

  1. 在平台首页点击"上传数据集"按钮或在左侧导航栏选择"数据集管理"
  2. 在数据集管理页面，点击"新建数据集"按钮
  3. 在弹出的对话框中，选择上传方式：
    • 本地文件上传
    • 远程URL导入
    • 选择内置数据集
  4. 根据选择的上传方式，按照提示完成上传操作
  5. 设置数据集参数（如下所述）
  6. 点击"确认上传"按钮
// 前端数据集上传示例
async function uploadDataset() {
  const formData = new FormData();
  formData.append('file', datasetFile);
  
  const response = await fetch('/api/datasets/upload', {
    method: 'POST',
    body: formData
  });
  
  return await response.json();
}

数据集参数设置

上传数据集时，您需要设置以下参数：

  • 数据集名称：为数据集指定一个易于识别的名称
  • 数据类型：指定数据集的类型（图像、表格等）
  • 标签信息：提供类别标签或目标变量信息
  • 预处理选项：选择是否需要进行标准化、归一化等预处理
  • 数据分割：设置训练集、验证集和测试集的比例

提示

对于图像数据集，系统会自动提取图像特征，您只需提供标签信息即可。

管理数据集

查看数据集

上传完成后，您可以在"数据集管理"页面查看所有已上传的数据集：

  1. 点击数据集名称可查看详细信息
  2. 数据集详情页面显示数据分布、特征统计等信息
  3. 可以预览数据样本，了解数据集内容

编辑和删除

您可以对已上传的数据集进行以下操作：

  • 编辑信息：修改数据集名称、描述等元数据
  • 更新数据：上传新版本的数据集
  • 删除数据集：从系统中移除不再需要的数据集

使用API上传

除了Web界面，您还可以使用API以编程方式上传数据集：

# 使用Python API上传数据集
from ml_security import MLSecurityClient

# 初始化客户端
client = MLSecurityClient(api_key="your_api_key")

# 上传数据集
dataset_id = client.upload_dataset(
    file_path="path/to/dataset.csv",
    name="My Dataset",
    description="Description of the dataset",
    data_type="tabular"
)

print(f"Dataset uploaded with ID: {dataset_id}")
