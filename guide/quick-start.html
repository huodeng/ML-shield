快速开始

本章将指导您快速上手ML Security Simulator平台，只需三个简单步骤，即可开始使用我们的平台进行机器学习模型安全评估。

准备工作

在开始使用平台前，请确保您已准备好以下内容：

  • 写好的机器学习模型（支持PyTorch格式）
  • 用于评估的数据集
  • 浏览器（推荐使用Chrome或Firefox最新版本）
# 机器学习模型示例代码
# 定义网络模型
import torch.nn as nn
import torch.nn.functional as F
class M(nn.Module):
    def __init__(self):
        super(M, self).__init__()
        self.conv1 = nn.Conv2d(1, 6, (5, 5), (1, 1), 2)
        self.conv2 = nn.Conv2d(6, 16, (5, 5))
        self.fc1 = nn.Linear(16 * 5 * 5, 120)
        self.fc2 = nn.Linear(120, 84)
        self.fc3 = nn.Linear(84, 10)
    def forward(self, x):
        x = self.conv1(x)
        x = F.relu(x)
        x = F.max_pool2d(x, 2, 2)
        x = self.conv2(x)
        x = F.relu(x)
        x = F.max_pool2d(x, 2, 2)
        x = x.view(x.size(0), -1)
        x = self.fc1(x)
        x = F.relu(x)
        x = self.fc2(x)
        x = F.relu(x)
        x = self.fc3(x) # 计算分类后每个数字的概率值
        return F.log_softmax(x, dim=1)

步骤一：上传数据集

首先，您需要上传用于评估的数据集：

  1. 在平台首页点击"上传数据集"按钮
  2. 选择要上传的数据集文件
  3. 设置数据集参数（如像素大小等）
  4. 点击"确认上传"按钮

数据集目录结构示例：

images/
├── train/
│   ├── class1/
│   │   ├── img1.jpg
│   │   ├── img2.png
│   │   └── ...
│   └── class2/
│       └── ...
└── test/
    └── ...

使用压缩文件(.zip)上传，系统会自动解压并处理。 :::

步骤二：选择分析任务

上传数据集后，您可以选择要执行的安全分析任务：

  1. 在左侧导航栏选择"模型分析"
  2. 从分析类型中选择需要的任务（如成员推理等）
  3. 设置分析参数
  4. 点击"开始分析"按钮

步骤三：查看分析结果

分析完成后，您可以查看详细的分析结果：

  1. 在结果页面查看攻击成功率，训练集准确率，测试集准确率等指标
  2. 查看可视化图表，了解攻击效果
  3. 导出分析报告，保存分析结果

注意

攻击分析可能需要较长时间，特别是对于大型模型和数据集。请耐心等待分析完成。
